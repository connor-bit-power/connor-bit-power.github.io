Stable Diffusion Image Generator for OpenAgents/GPUTopia

Description

Built an interface that loads & connects the SDXL model to the GPUTopia decentralized gpu cluster via FastAPI. React front end allows users to pay gpu providers in Bitcoin to send requests to gpu cluster and recieve completed SDXL images.

Tech Used

Python, FastAPI, Stable Diffusion, React, Radix-UI

YEWS Desktop for YEWS

Description

YEWS is a news site built for mobile. YEWS Desktop is a unique experience that allows users to view and interact with the daily YEWS through their browser, with the 3D calendar being the main method of interaction. Data is gathered and indexed in real time from a custom api accessing yews.news, as well as other relevant real time data such as the price of Bitcoin and the population of Earth.

Tech Used

ThreeJS, FastAPI, React
