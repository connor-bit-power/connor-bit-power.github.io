YEWS Desktop (RAG Chatbot)

Description

YEWS is a news site perfectly built for mobile. YEWS Desktop is a unique experience that allows users to view and interact with the daily YEWS through their browser, with the 3D calendar being the main method of interaction. Data is gathered and indexed in real time from a custom api accessing yews.news, as well as other relevant real time data such as the price of Bitcoin and the population of Earth.

Tech Used

Frontend: React, ThreeJS

Backend: Python, FastAPI, BeautifulSoup

Stable Diffusion Image Generator

Description

App built for GPUtopia that loads and connects the SDXL model to the GPUtopia decentralized gpu cluster via FastAPI. React front end allows users to pay gpu providers in Bitcoin to send requests to gpu cluster and receive completed SDXL images.

Backend: Python(FastAPI), HuggingFace API, Stable Diffusion

Home Lab
Llama.cpp server

Description

Currently building/tweaking for personal use, need to take stuff out before posting. Llama.cpp is an open source framework for locally running open source LLMs.
The goal of the Llama.cpp server is to have a simple server that uses Llama.cpp and Op*nAI request/response framework to load and interact with open source LLMs running on their local machine or home server. I've been purpose building it for myself, in a modular way that allows me to plug in new models, add funcitonality like TTS for controlling the server through voice and plug in whatever API services I want to use.

Tech Used

Llama.cpp, Python, C++, FastAPI, NeetsAPI
